# 📝 Todo API Backend

This is a simple RESTful API for managing todos, built using **Node.js**, **Express**, and **MongoDB**.

## 🚀 Features

- Create a new todo
- Get all todos
- Update a todo
- Delete a todo

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB (local or cloud)
- Mongoose
